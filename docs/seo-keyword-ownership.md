# SEO Keyword Ownership

This map prevents keyword cannibalization by assigning one primary intent per landing route.

## Single Source Of Truth

| URL | Primary keyword | Secondary keywords | Search intent | Owner page |
| --- | --- | --- | --- | --- |
| `/` | queer atlas | queer travel, lgbtq travel | brand discovery | `/` |
| `/gay-guide` | gay travel guide | gay guide, gay travel | informational travel guide | `/gay-guide` |
| `/queer-guide` | queer travel guide | queer travel, lgbtq travel guide | informational travel guide | `/queer-guide` |
| `/hbtq-guide` | hbtq guide | hbtq travel, hbtq stader | regional-language guide intent | `/hbtq-guide` |
| `/cities` | gay travel cities | queer cities, lgbtq city guides | category discovery | `/cities` |
| `/events` | queer events | gay events, lgbtq events | time-based discovery | `/events` |
| `/[city]` | gay bars in {city} | gay clubs {city}, queer bars {city}, lgbt nightlife {city}, gay sauna {city}, queer events {city} | local nightlife and venue discovery | `/[city]` |

## Section Usage Rule

Use cluster terms naturally in city subheadings and section intros, not as repeated lists. One clear mention per section is enough.

## Rule

Each landing route owns one primary keyword cluster. Other routes should reference, not compete with, that exact primary phrase.
