# QueerAtlas BA3: Backlink + Outreach Sprint (20 Targets)

## Scope
- Goal: increase entity authority and AI citability by earning mentions/links from trusted LGBTQ+ and travel publishers.
- Rule: pitch report URLs first (`/reports/...`), then attach one city/topic route as proof.
- Window: 14 days.

## Operating Rules
- Use one target, one clear ask, one primary URL.
- Do not mass-email from one template without personalization.
- Track every send and response in one sheet.
- Prefer editorial value over self-promotion language.
